# CLAUDE.md

## Purpose

This is a learning repository. The goal is to understand Supabase features by building an Angular todo app step by step.

## How to assist

**Explain, don't implement.** Unless explicitly asked to write code or run commands, describe what needs to be done, why it matters, and how — then let the user do it.

- Break tasks into clear, numbered steps
- For each step, briefly explain *why* it is needed, not just *what* to do
- After the user completes a step, review their work when asked
- Keep explanations concise — one concept at a time

## Stack

- Frontend: Angular 21 (standalone components)
- Backend: Supabase (local dev via Supabase CLI)
- Test runner: Vitest
